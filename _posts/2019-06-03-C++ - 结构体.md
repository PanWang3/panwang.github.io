- 结构体是一个构造类型
- 结构体变量的声明方式

## 声明

- 方式1: 在定义结构体后，使用结构体类型名声明

```
struct PersonInfo
{
    int index;
    char name[30];
    short age;
};

PersonInfo pInfo;
```

- 方式2:定义结构体时直接声明，直接声明时，可以声明多个变量

```    
struct PersonInfo
{
    int index;
    char name[30];
    short age;
} pInfo1, pInfo2;
```

## 初始化

- 方法一： 赋值

```
#include <iostream>

pInfo.index = 0;
pInfo.age = 30;
strcpy(pInfo.name,"Geroge");
```

- 方法二：定义时直接赋值

```
struct PersonInfo
{
    int index;
    char name[30];
    short age;
} pInfo = {0,"George",30};
```
